# PDF RAG LLM Assistant

A Retrieval-Augmented Generation (RAG) system that allows users to query PDF documents using a Large Language Model.

## Features
- PDF text extraction
- Chunking & embeddings using FAISS
- Semantic search
- LLM-based answer generation (Gemini)

## Tech Stack
- Python
- FAISS
- Google Gemini API
- PyPDF2
